Hexo Admin

  1. 安装 hexo-admin 插件 在使用 npm 安装模块时，可以在 npm install 命令后加上 –save 或 –save-dev 参数，来将模块添加到项目的 package.json 文件中的 dependencies 或 devDependencies 字段。

–save 参数表示将模块添加到 dependencies 字段中，这意味着该模块是项目的必要依赖项，并且会在项目部署时被安装和使用。

  1. 修改根目录中的 _config.yml 文件
    启用hexo-admin插件 以下设置在登录进Admin后 可以初始化设置
    在localhost:4000/admin/网站中点击右上角的settings，然后点击下面的 Setup authentification here
  2. 生成管理员密码的哈希值 3-2. 安装 bcryptjs 模块
  3. 启动Hexo本地服务器
  4. 访问 http://localhost:4000/admin/ ，您将看到一个登录页面。输入您在 _config.yml 文件中设置的用户名和密码，即可登录管理界面。
