★Bootstrap 5.2.3 两种 JS
在 Bootstrap 5.2.3 中，有两个主要的 JavaScript 文件可用：bootstrap.min.js 和 bootstrap.bundle.min.js。它们之间存在以下区别：

bootstrap.min.js：这是 Bootstrap 的核心 JavaScript 文件，仅包含 Bootstrap 组件所需的 JavaScript 功能。如果你的项目中使用的是其他库（如 jQuery），并且已经将其导入到项目中，那么你可以只使用 bootstrap.min.js 文件，它将自动与你的库进行集成。

bootstrap.bundle.min.js：这是一个更全面的 JavaScript 文件，它包含 Bootstrap 的核心 JavaScript 功能以及一些依赖的第三方库，例如 Popper.js（用于处理弹出窗口和工具提示的定位）。默认包含的 jQuery 版本是 jQuery 3.6.0。

所以，如果你的项目中没有导入其他库（如 jQuery 和 Popper.js），并且你希望使用 Bootstrap 的全部功能，包括弹出窗口和工具提示的定位等功能，那么建议使用 bootstrap.bundle.min.js。它是一个单一的文件，包含了所有所需的依赖，方便使用和部署。

如果你已经导入了其他库或只需使用 Bootstrap 的组件功能，你可以使用较小的 bootstrap.min.js 文件。
